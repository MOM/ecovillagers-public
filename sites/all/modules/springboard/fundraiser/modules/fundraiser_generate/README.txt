INTRODUCTION
------------

Current Maintainer: Jackson River http://jacksonriver.com

This module generates donations for testing purposes. Donations may be
generated in one mass lump, or configured to be recurring for testing over
time.

INSTALLATION
------------

1. Unzip and copy this directory to your sites/SITENAME/modules directory. 

2. Enable the module. 

3. Configure user permissions at admin/user/permissions.

4. Go to admin/config/development/generate/donation to generate donations.

DESCRIPTION
-----------
This module provides integration with Sage Payment Solutions for Drupal Commerce.

Note that Sage Payment Solutions (http://na.sage.com/sage-payment-solutions) is
the North American solution and distinct from the Sagepay solution
(http://www.sagepay.co.uk/) used in Europe.

INSTALLATION
------------

The cardonfile functionality requires commerce_cardonfile 2.x.

Documention for the Sage Payment XML API can be downloaded from.
https://www.sagepayments.net/developer/
